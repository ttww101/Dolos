## 此專案可以幹嘛?

[x] 替換 File 上方描敘檔的創建日期跟創建者
[x] 產生 Objective-C Spam Code
[x] 產生 Swift Spam Code
[x] 修改圖片名稱
[x] 刪除 File 上方描敘檔
[x] 支援忽略路徑
[x] 活躍的社群
[ ] 更改檔案名稱

## 參數說明


|  參數名 |   解釋  |  備註 |
|---|---|---|
|   -spamCode   |   產生垃圾 code     | |
|   -handleXcassets    |   修改圖片名稱    |  |
|   -replaceFileDescription    |   替換 File 上方描敘檔的創建日期跟創建者   |  不可跟-deleteComments 同時使用 |
